# Rust Learning Summary

- **Day 1**: Rust mental model, basic syntax, mutability, stack vs heap, and pattern matching.
- **Day 2**: Ownership laws, borrowing, references, and the borrow checker.
- **Day 3**: Structs, Enums (ADTs), and error handling using `Option` and `Result`.
- **Day 4**: Traits, Generics, trait bounds, and the architecture of Actix extractors.
- **Day 5**: Cargo package management, project structure, and the Rust module system.
- **Day 6**: Async Rust, the Tokio runtime, and building a basic Actix Web server.
